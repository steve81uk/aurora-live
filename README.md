# Aurora-live

Real-time aurora visualisation with live NOAA space weather data and location-based visibility predictions.

## Overview

Aurora-live provides real-time visualizations of aurora borealis and australis activity using live data from NOAA's space weather monitoring services. Users can see current aurora predictions based on their location and get alerts for optimal viewing opportunities.

## Features

- Real-time aurora activity data from NOAA
- Location-based visibility predictions
- Interactive visualization of aurora forecasts
- Real-time updates and alerts

## Getting Started

### Prerequisites

- Node.js 16.x or higher
- npm or yarn

### Installation

```bash
npm install
```

### Running the Application

```bash
npm start
```
